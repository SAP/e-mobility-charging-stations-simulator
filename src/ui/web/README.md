# Web UI

## Project setup

```
npm install
```

### Run

For both solution you can then follow the link displayed in the terminal at the end of compilation

#### Compiles and run for production

```
npm start
```

#### Compiles and run for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints files

```
npm run lint
```
